# ELEVATE-LABS-INTERSHIP
# Python Developer Internship Tasks

## Task 1: Calculator CLI App
A command-line calculator that supports:
- Add
- Subtract
- Multiply
- Divide
- Loop until user exits

Run:
python calculator.py

## Task 2: To-Do List CLI App
A persistent console-based to-do list.
Features:
- Add tasks
- View tasks
- Remove tasks
- Stores tasks in tasks.txt

Run:
python todo.py

## Concepts Used
- Functions
- Loops
- Conditionals
- File Handling
- Lists
- String manipulation
